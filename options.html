cyberTWEAK Options

This extension is intended to thwart social engineering attacks by spoofing the user agent string that your browser sends with each website request. The user agent string contains info such as your browser version and operating system, spoofing it can conceal your identity and mislead potential attackers.

How to use

Enter the domain of each website for which you'd like to change your user agent sting. You do not need to include "http://"" or "www.". You may enter subdomains ("support.google.com") or specific paths ("google.com/maps"), but do not add a trailing "/".

You can choose your desired User Agent String below as well as a probabilities for which each domain will be changed (percentage 0-100). After setting your estimated visits/week for each website, you can run the algorithm which will set the percentages for each website and estimate each website's risk.

As altering the user-agent string can degrade the webpage rendered, you can enter a budget for total degradation due to traffic alteration. Enter a number 0-1000, the default is 500.
