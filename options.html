Anti-Fingerprinting Options

About this Extension

This extension was developed at Carnegie Mellon University and is based on research by Ryan Shi, Aaron Schlenker, Brian Hay, Aaron Ferber, and Fei Fang

How to use

Enter the domain of each website for which you'd like to change your user agent sting. You do not need to include "http://"" or "www.". You may enter subdomains ("support.google.com") or specific paths ("google.com/maps"), but do not add a trailing "/".

You can also choose your desired User Agent String for each domain as well as a probability for which that domain will be changed (percentage 0-100).
